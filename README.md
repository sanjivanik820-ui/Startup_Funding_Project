**Startup Funding Analysis Project Report**

**Project Title:**
Indian Startup Funding Analysis Dashboard

**Objective:**
The objective of this project is to analyze Indian startup funding data and create an interactive dashboard that helps users understand funding trends, investor activities, startup performance, sector growth, and city-wise investment patterns.

**Tools & Technologies Used**
Python
Streamlit
Pandas
NumPy
Plotly Express
CSV Dataset

**Project Description:**
This project is a web-based data analytics dashboard developed using Streamlit. It processes startup funding data, performs data cleaning and transformation, and presents insights through interactive visualizations.

**Users can explore:**
Overall funding trends
Startup-wise analysis
Investor-wise analysis
Sector performance
City-wise funding distribution

**Dataset Features:**
The dataset contains information such as:
Startup Name
Industry Vertical
Sub Vertical
City Location
Investor Name
Funding Amount
Funding Round
Funding Date

**Data Preprocessing:**
The following preprocessing steps were performed:

Removed unnecessary columns.
Renamed columns for better readability.
Converted date fields into datetime format.
Extracted Year and Month from funding dates.
Cleaned funding amount values by removing commas and special characters.
Standardized investor names.
Standardized startup names.
Converted funding amounts into numeric format.

**Dashboard Modules**

**1. Overall Analysis:**
Provides a high-level overview of startup funding.

**Features:**
Total Funding Raised
Maximum Funding Received
Average Funding per Startup
Total Number of Startups
Monthly Funding Trend
Sector-wise Funding Analysis
City-wise Funding Analysis
Top Funded Startups
Top Investors

**3. Startup Analysis**
Allows users to select a startup and view detailed information.

**Features:**
Industry Information
Location Details
Funding History
Total Funding Raised
Similar Startups in the Same Industry

**4. Investor Analysis**
Allows users to analyze investor behavior.

**Features:**
Recent Investments
Largest Investments
Preferred Industry Sectors
Investment Distribution Charts

**Visualizations Used**

**Line Chart:**
Used to display month-on-month funding trends.

**Pie Chart:**
Used for:
Sector-wise funding distribution
Investor sector preferences

**Bar Chart:**
Used to compare city-wise funding amounts.

**Data Tables:**
Used to display:

Top startups
Top investors
Funding rounds
Investment records

**Key Insights**
Major startup hubs attract the highest investments.
Technology and e-commerce sectors receive significant funding.
Certain investors consistently invest in specific sectors.
Funding activity varies across years and cities.
Top startups account for a large percentage of total investments.

**Benefits of the Project**
Helps investors identify funding trends.
Assists startups in understanding investment patterns.
Enables data-driven decision-making.
Provides quick access to startup ecosystem insights.

**Future Enhancements**
Add machine learning-based funding prediction.
Include startup success probability analysis.
Integrate real-time funding data through APIs.
Add advanced filtering and search options.
Deploy dashboard on cloud platforms.

**Conclusion:**
The Startup Funding Analysis Dashboard successfully transforms raw startup funding data into meaningful 
business insights through interactive visualizations. The project demonstrates skills in data cleaning, exploratory data analysis, 
dashboard development, and data visualization using Python and Streamlit.
